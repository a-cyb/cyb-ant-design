# cyb ant design

基于塞伯坦和antDesign的最佳工程实践解决方案

## 基本特性

* 集成最佳响应式布局的基本骨架。
* 集成路由配置的最佳实践。
* 集成状态管理的最佳实践。
* 集成异步编程及数据处理的最佳实践。
* 集成Mock数据和联调方案的最佳实践。

## 使用

```
npm install
npm start
```

## More

更多命令功能请参考 `package.json` 的 `scripts`。
